# gRPC vs gomobile binding

gRPC と gomobile bind の速度比較のためのリポジトリです。

## ルール

* Java 側から Go 側の API を呼び出して string の配列を取得する。
  * 1000 要素ある string (256 byte 長) の配列を 1000 回取得して速度を測定する。

* Go バインディングは文字列の配列を返却できないので、JSON Array で代替する。
* gRPC は 文字列の配列を返す。

* 文字列の配列は以下。
